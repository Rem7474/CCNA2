# Outils (tools)

Ce dossier contient des scripts utilitaires utilisés pour préparer et convertir les données de quiz.

Fichiers présents
- `csv_to_json.py` — Convertit le fichier CSV des questions (`ExamCisco2.csv`) en JSON (`data/final.json`).
- `scrape_ccna.py` — Script heuristique pour récupérer des questions depuis une page web et produire un brouillon JSON dans `data/`.

But
---
Documenter rapidement l'utilisation et les dépendances des scripts afin de faciliter leur exécution locale.

Voir les fichiers dédiés pour chaque script :
- `csv_to_json.md` — instructions pour `csv_to_json.py`.
- `scrape_ccna.md` — instructions pour `scrape_ccna.py` (attention aux limitations légales et techniques).
